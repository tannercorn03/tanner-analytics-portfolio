# Friday Night Football Defensive Analytics

Partnered with a high school football coaching staff to analyze opponent offensive tendencies and support defensive game planning using data.

## Objective
Identify trends in opponent formations, play types, and situational tendencies to help coaches make data-driven defensive adjustments.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data cleaning and tracking

## What I Did
- Collected play-by-play data for each opponent
- Tracked down, distance, formation, run/pass, yards gained, and tackler
- Cleaned and organized data into structured spreadsheets
- Built pivot tables to analyze:
  - Run vs pass tendencies
  - Formation effectiveness
  - Yards allowed per play

## Results & Insights
- Highlighted opponent tendencies by formation and down/distance
- Provided summaries to coaching staff before games
- Helped inform defensive strategy and preparation

# Bike Share Customer Analysis (Cyclistic Case Study)

Capstone project completed as part of the Google Data Analytics Professional Certificate. Analyzed how casual riders and annual members use a bike-share service differently to recommend strategies for increasing memberships.

## Objective
Understand behavioral differences between rider types and identify opportunities to convert casual riders into annual members.

## Tools Used
- Microsoft Excel
- Data cleaning
- Pivot tables
- Data visualization

## What I Did
- Cleaned and prepared a large ride-history dataset
- Analyzed ride length, day of week, and time of day trends
- Compared usage patterns between casual and member riders
- Built pivot tables and summary metrics
- Created visualizations and a presentation of findings

## Results & Insights
- Identified peak usage differences between customer types
- Highlighted opportunities for targeted marketing
- Presented actionable recommendations to improve member conversions
